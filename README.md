Ghibli is a simple and elegant iOS movie recommendation app built with SwiftUI. It fetches Studio Ghibli film data, displays rich movie details, and allows users to explore titles effortlessly through a clean, modern interface. Designed as a learning project to practice networking, JSON decoding, and SwiftUI architecture.
